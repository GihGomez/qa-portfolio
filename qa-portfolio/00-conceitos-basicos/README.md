
# Conceitos Básicos — Baseado em ISTQB Foundation

## Tipos de Teste
- Funcionais (positivo/negativo)
- Não Funcionais (desempenho, segurança, usabilidade, acessibilidade)
- Regressão, Smoke, Sanity, Exploratório

## Níveis de Teste
- Unitário → Integração → Sistema → Aceitação (UAT)

## Abordagem por Risco
- Priorizo fluxos críticos e requisitos de alto impacto no negócio.

## Rastros (Traceability)
- Requisito → Caso de Teste → Execução → Bug → Evidência → Métrica
