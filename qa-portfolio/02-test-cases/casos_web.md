
# Casos de Teste — Login & Checkout

## CT-LOGIN-001 — Login válido (Crítico)
**Passos**: acessar login → usuário válido → senha válida → Login.  
**Esperado**: redireciona para listagem de produtos.

## CT-LOGIN-002 — Senha inválida (Alto)
**Esperado**: mensagem clara e permanece na tela de login.

## CT-CHK-001 — Finalizar compra com 1 item (Crítico)
**Esperado**: pedido confirmado com número visível.

## CT-CHK-002 — CEP inválido (Médio)
**Esperado**: validação específica de CEP.
