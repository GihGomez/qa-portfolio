
# Técnicas de Projeto de Teste

## Partição de Equivalência (EP)
- Ex.: CEP: valores válidos (somente dígitos) vs. inválidos (letras/especiais).

## Análise de Valor Limite (BVA)
- Ex.: Nome: 0, 1, 2, 50, 51 caracteres.

## Tabela de Decisão
- Ex.: Regras de frete grátis por faixa de preço e região.

## Pairwise (combinatório)
- Ex.: Navegador × SO × Resolução — selecionar combinações mínimas com máxima cobertura.
